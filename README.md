# Web3 School for Newbies

A beginner-friendly AI skill that teaches Web3 concepts and safe on-chain interaction step by step.

## What this is
This project helps complete beginners understand Web3 without overload. It routes users by intent and teaches one concept at a time.

## Key idea
Instead of explaining everything at once, it:
- identifies user goal
- teaches one concept at a time
- gives small practice steps
- gradually unlocks deeper topics

## User paths
- Concept learners
- Wallet/DApp users
- NFT / DeFi / airdrop explorers
- Developers

## Usage
Use SKILL.md in agent systems that support skills.
Use Universal Prompt in systems without skill support.

## Safety
No financial advice. No private key requests. No guaranteed returns.
