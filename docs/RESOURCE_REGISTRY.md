# Resource Registry

This file lists external resources that can be released gradually based on the learner's path.

The assistant should not dump this whole list into a beginner conversation. Use one or two resources at a time.

Last reviewed: 2026-06-16

## Concept and General Beginner Resources

### Ethereum.org Learn

URL: https://ethereum.org/en/learn/

Use for:

- Web3 concept basics
- Ethereum basics
- wallets
- gas
- transactions
- security
- quizzes and beginner guides

Best for:

- Concept Explorer
- Wallet and DApp Operator

### MetaMask Learn

URL: https://learn.metamask.io/

Use for:

- wallet basics
- self-custody
- Web3 introduction
- NFT / DeFi beginner concepts
- wallet safety

Best for:

- complete non-technical beginners
- wallet operation users

### Binance Academy

URL: https://academy.binance.com/

Use for:

- crypto and blockchain vocabulary
- beginner and intermediate tracks
- on-chain analysis introduction

Best for:

- broad beginner education

## Resource Release Rules

### For concept beginners

Release:

1. Ethereum.org Learn
2. MetaMask Learn

### For wallet operators

Release:

1. MetaMask Learn
2. relevant block explorer
3. testnet faucet based on chosen network

### For earning-related learners

First teach:

1. connect wallet
2. sign
3. send transaction
4. approve
5. revoke

Then mention quest platforms as examples, not recommendations.

### For builders

Release in order:

1. Remix IDE
2. CryptoZombies or WTF Solidity
3. Alchemy University or Cyfrin Updraft
4. Scaffold-ETH
5. Speedrun Ethereum
6. Ethernaut
