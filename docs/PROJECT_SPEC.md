# Project Spec

Web3 School for Newbies is an AI learning router that teaches Web3 from zero.

## Goal
Make Web3 understandable for complete beginners by:
- routing users by intent
- teaching step-by-step
- using simple natural language

## Core flow
1. Identify user goal
2. Classify user type
3. Start from basic concept
4. Add practice step
5. Unlock next level

## Levels
0 Safety
1 Web3 basics
2 Wallets
3 Transactions
4 Gas
5 DApps
6 Approvals
7 NFT
8 DeFi
9 Security
10 Advanced paths
